Cafe Kiosk

An interactive coffee ordering application. The application allows the user to add drinks to cart, remove drinks from cart, view order summary, place order and generate bill. The application also loggs and keeps track of when a drink is added or removed from the cart. I implemented the JSON reader and writer classes to add saving and loading functionality. I used the Java Swing library to add a graphical user interface to make the application more user friendly. The application was written with software design principles in mind and tested using JUnit.

Languages: Java

Image Partitioning Trees

Constructed a PTree(a binary tree whose nodes represent rectangular regions of a PNG image). Implemented the render function to contruct the PNG. Implemented the Prune function which attempts, starting near the top of a freshly built tree, to remove all of the descendants of a node, if all of the leaf nodes below the current node have colour within tolerance of the node's average colour. Pruning degrades the image quality. Finally, I implemented two additional functions to flip the PTree horizontally and vertically.

Languages: C++
